## 项目概述

本项目是一款基于语音识别的智能垃圾分类系统，旨在通过自然语言处理技术，实现对普通话语音的识别，以帮助用户正确分类垃圾。

## 功能模块

主要功能如下：

1. 语音录制与上传：用户可录制WMV格式的语音文件并上传至系统。
2. 语音识别与分析：系统采用自然语言处理方式分析语音内容，如播放“废旧电池”等词汇，以识别垃圾类别。
3. 智能提示：当识别完成后，对应的分类垃圾桶会给出相应的提示，如打开盖子等。
